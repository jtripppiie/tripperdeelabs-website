About the lab

Independent software for curious people.

TripperDeeLabs is a small independent software studio creating Android apps, Firefox extensions, and browser-based experiments. The lab focuses on practical tools with a playful edge—software that helps people explore nearby places, understand the web, or solve an oddly specific problem.

Our projects include Local Lore, a privacy-conscious local discovery app for history, wildlife, geology, legends, and curious facts; Firefox developer tools for inspecting layouts and hard-to-catch hover states; and experimental utilities for screenshots, data extraction, and everyday browsing.

Every TripperDeeLabs project is built with clear privacy choices, straightforward explanations, and respect for the person using it. We favor useful features over unnecessary accounts, tracking, and clutter.
